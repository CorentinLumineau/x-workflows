## Layer Responsibilities

| Layer | Purpose | Files |
|-------|---------|-------|
| API Gateway | Auth, rate limiting, logging | middleware/ |
| Routes | HTTP endpoints | routes/, handlers/ |
| Services | Business logic | services/ |
| Repositories | Data access | repositories/, models/ |
| Database | Data storage | migrations/, schema |
